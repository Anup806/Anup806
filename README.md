## 🚀 Featured Projects

### 🧮 [Adaptive Mixed-Reality Math Tutor](https://github.com/Anup806/Inclusive-Mixed-Reality-Math-Experience-design-Using-ML-Based-Models)
> **Final Year Project** — Inclusive AR platform combining gesture recognition, adaptive ML, and real-time feedback for personalized math education

- ✋ Built gesture-controlled drag-and-drop interaction using **MediaPipe** 21-point hand landmark detection and **OpenCV** pinch gesture recognition
- 📊 Implemented **Bayesian Knowledge Tracing (BKT)** to model per-student mastery probability (`p_know`) in real time
- 🔬 Applied **PC-BKT K-Means clustering** on 5-feature vectors (accuracy, p_know, response time, consistency, engagement) to classify students into BASIC / INTERMEDIATE / ADVANCED tiers
- 💾 Built session-persistence pipeline logging every interaction to CSV for longitudinal analytics and model retraining
- ♿ Designed multimodal feedback (visual animations + audio counting) for accessibility and inclusivity

**Tech Stack:** `Python` `OpenCV` `MediaPipe` `scikit-learn` `NumPy` `Pandas` `Pygame`
